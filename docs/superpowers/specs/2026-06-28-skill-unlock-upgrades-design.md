# 猪猪黄昏：技能升级解锁设计

## 目标

将主动技能纳入单局升级成长。开局保留屠刀旋斩，盐袋爆撒和猪圈陷阱需通过升级三选一获得。

## 开局状态

- `Q` 屠刀旋斩默认解锁。
- `E` 盐袋爆撒默认锁定。
- `R` 猪圈陷阱默认锁定。
- 技能解锁状态只属于当前单局，不写入长期存档。

## 升级池规则

- 新增“盐袋爆撒”和“猪圈陷阱”解锁卡。
- 未解锁技能的强化卡不得出现。
- 技能解锁后，对应解锁卡移出候选池，对应强化卡加入候选池。
- 普通属性卡始终可进入候选池。
- 金币刷新使用同一过滤规则，不得绕过解锁前置。
- 三个选项尽量不重复；候选数量不足时才允许循环填充。

## 解锁与强化

- 选择解锁卡后，技能立即可用，冷却从零开始。
- 盐袋爆撒解锁后，“粗盐增量”才能出现。
- 猪圈陷阱解锁后，“猪圈加固”才能出现。
- “火候正好”作为已解锁主动技能的通用强化，至少解锁盐袋爆撒后才可出现。

## HUD 与交互

- 未解锁的 `E` / `R` 技能格降低亮度，显示锁定标记和“升级时获得”。
- 未解锁技能不显示“就绪”，键盘与鼠标点击都不会施放。
- 解锁时播放一次简短的技能格高亮反馈。
- 升级卡明确区分“解锁技能”和“强化技能”，描述实际效果。

## 数据与边界

- `PlayerProgressState` 保存当局已解锁技能集合。
- `UpgradeSystem` 根据玩家成长状态构建有效候选池。
- `Player.tryCast` 在检查冷却前先检查解锁状态。
- HUD 状态包增加技能解锁信息，不依赖 HUD 自行推断。
- 暂不增加新技能、技能槽数量或跨局技能解锁。

## 验收

- 新局只能施放旋斩。
- 升级可抽到盐袋爆撒和猪圈陷阱解锁卡。
- 选择解锁卡后对应技能立即可用。
- 未解锁技能不会抽到对应强化。
- 刷新、重开和继续游戏都不会破坏单局解锁规则。
