# 猪猪黄昏 三类持武模板设计

- 日期：2026-06-28
- 主题：角色持武表现重做
- 范围：按武器类型重做角色持武模板，替换当前通用武器定位方案

## 1. 背景

当前运行时采用“角色 sprite + 独立武器 sprite”组合方案，并通过通用偏移、旋转和前伸距离来摆放武器。这个方案已经暴露出结构性问题：

- 武器虽然被定位了，但看起来并没有真正被角色拿在手里
- 左右朝向和攻击朝向容易反
- 不同外形武器差异过大，单一通用定位公式无法稳定覆盖
- 越继续为单把武器补偏移，越容易形成脆弱且难维护的特例堆积

本次调整不再把问题当成“个别坐标不对”，而是直接把角色持武表现重做为按武器类型划分的模板系统。

## 2. 目标

本次改动目标如下：

- 让角色在待机、走路、攻击时都明显表现为“手里拿着武器”
- 用三类持武模板覆盖当前保留的 7 把武器
- 保留单把武器少量微调能力，但不再依赖单武器硬编码撑起整体表现
- 让左右朝向、攻击起点和武器朝向符合直觉

本次不做：

- 为每把武器绘制完全独立的一整套角色动画
- 重做攻击判定系统
- 重新引入已删除的流星锤和枪
- 修改地图、敌人、HUD 或升级系统

## 3. 三类模板

### 3.1 轻刀类

包含：

- `rustyCleaver`
- `saltFrostCleaver`
- `pigBoneChainsaw`

表现目标：

- 单手持握
- 待机时武器贴近身体侧前方
- 走路时保持轻微跟手摆动
- 攻击时短促斜砍，不做大幅甩出

### 3.2 重刀类

包含：

- `dragonSlayer`
- `hundredMeterPigBlade`

表现目标：

- 更靠近身体中线的重持姿态
- 待机时武器压低，体现重量
- 走路时摆动较小，避免轻飘
- 攻击时以前压重劈为主

### 3.3 钩钳类

包含：

- `duskHook`
- `hydraulicBoneCrusher`

表现目标：

- 武器横向或斜向贴近身体前侧
- 待机时强调工具感而不是刀感
- 攻击时前探钩击或夹击
- 保持机械感，但依然要像是被角色抓在手里

## 4. 实现策略

### 4.1 替换共享定位公式

不再只使用当前一套共享的：

- 中心点起始
- 通用前伸距离
- 通用旋转幅度

改为：

- 先按武器类型选择持武模板
- 模板定义待机手位、移动手位、攻击起止角度、攻击前伸和深度规则
- 单把武器仅保留少量微调参数

### 4.2 模板字段

每个模板至少要提供：

- `idleHandOffset`
- `moveHandOffset`
- `attackHandOffset`
- `idleRotation`
- `attackStartRotation`
- `attackEndRotation`
- `idleReach`
- `attackReach`
- `depthBias`

单把武器可额外提供轻量 `fineTune`：

- `x`
- `y`
- `rotation`
- `scale`

### 4.3 角色表现

本次不要求新增完整手绘逐帧动画资源；优先使用当前角色动画基础，通过更可靠的持武模板让视觉成立。

如果现有角色动画与持武模板有局部冲突，可以在以下范围内做轻量增强：

- 调整待机 pose 的武器挂点
- 调整 walk pose 的武器挂点
- 调整 attack pose 的武器挂点

核心原则是：角色动作继续尽量复用，但持武关系必须正确。

## 5. 数据结构

建议在现有武器数据之外新增两层：

- `weaponHoldTemplateCatalog`
- `weaponHoldOverrides`

其中：

- 模板目录负责定义三类公共行为
- 单武器覆盖只负责细节微调

武器自身仍保留：

- 稀有度
- 攻击力
- 攻速
- 攻击范围
- 攻击类型

但“如何拿在手上”不再由单一 `grip + worldScale + attackProfile` 直接推导。

## 6. 测试策略

需要新增或更新以下测试：

- 轻刀类待机持武测试
- 重刀类攻击持武测试
- 钩钳类待机/攻击测试
- 左右朝向镜像测试
- 单武器微调不会把武器推回角色中心外漂移的测试

验证目标：

- 武器起点来自角色手位，而不是角色中心
- 三类模板在左右朝向下都成立
- 同类武器共用模板时观感统一
- 单把武器仍可做有限微调

## 7. 风险与边界

主要风险：

- 如果只改模板、不观察真实页面，可能仍会出现视觉误差
- 某些武器贴图透明边界较大，可能需要覆盖偏移补偿
- 旧测试如果仍绑定旧定位公式，需要一起迁移

边界控制：

- 本次优先解决“角色像拿着武器”
- 不追求每把武器拥有完全独立专属演出
- 一旦三类模板在页面里成立，再考虑第二轮单武器精修
